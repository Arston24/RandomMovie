# RandomMovie
Позволяет сгенерировать случайный фильм, найти любой фильм в базе данных The Movie Database. <br/>
API_KEY находится в local.properties, можно получить на [themoviedb.org](https://www.themoviedb.org)

## Библиотеки
* Retrofit 2
* Gson
* Glide
* Room
* Coroutines
